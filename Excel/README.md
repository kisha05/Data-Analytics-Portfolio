# Excel Projects

This folder contains Microsoft Excel dashboards created using Pivot Tables, Pivot Charts, and Slicers for business analysis.

## 1. Sales Dashboard 2023

### Project Overview
Built an interactive sales dashboard in Microsoft Excel to analyze revenue across products, occasions, categories, and locations.

### Features
- Interactive Slicers
- Pivot Tables
- Pivot Charts
- Revenue Analysis
- Category-wise Analysis
- Product-wise Analysis
- Location-wise Analysis
- Occasion-wise Analysis
