# Deloitte Data Analytics Virtual Experience

This project was completed as part of the Deloitte Australia Data Analytics Virtual Experience Program on Forage.

## Project Overview

The project focused on analyzing operational data and creating interactive dashboards to generate business insights.

## Tasks Completed

- Analyzed machine telemetry data
- Cleaned and explored the dataset
- Created interactive Power BI dashboards
- Identified business insights from operational data

## Tools Used

- Power BI
- Microsoft Excel
- JSON Data
- Data Visualization
