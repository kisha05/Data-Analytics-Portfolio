# Deloitte Data Analytics Virtual Experience

This project was completed as part of the Deloitte Australia Data Analytics Virtual Experience Program on Forage.

## Project Overview

The project involved analyzing operational data and building dashboards to generate business insights.

## Tasks Completed

- Analyzed machine telemetry data
- Cleaned and explored the dataset
- Built interactive dashboards
- Generated business insights

## Tools Used

- Power BI
- Microsoft Excel
- JSON
- Data Visualization
