# Power BI Projects

This folder contains Power BI dashboards created for data visualization and business insights.

## Projects

### 1. Pizza Sales Dashboard

**Project Overview:**
Designed an interactive dashboard to analyze pizza sales performance, revenue, order trends, and customer purchasing patterns.

**Key Metrics:**
- Total Orders
- Total Revenue
- Total Quantity Sold
- Sales by Category
- Sales by Pizza Size
- Top Selling Pizzas

---

### 2. Student Performance Dashboard

**Project Overview:**
Built an interactive dashboard to analyze student academic performance and identify factors affecting final grades.

**Key Metrics:**
- Total Students
- Above Average Students
- Below Average Students
- Average Grade (G3)
- Performance by Study Time
- Performance by Internet Access
- Performance by Parents' Education
- Pass vs Fail Analysis
